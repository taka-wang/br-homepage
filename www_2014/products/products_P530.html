P530 Series
LTE WLAN Mobile Router

BandRich's P530 series mobile router has won the 2013 iF Product Design Award, a world recognized label for smart designs and supreme quality. BandRich would like to share its successful design and beauty with the world.

Impressive compact size, charming shape and streamlined glossy surface make you feel solid and comfort at hand. Simply touching on its ergonomic slider, the user may intuitively slide the top cover to turn on the device and LTE broadband connection will happen naturally.

With Li-ion battery and LTE/WiFi/USB high speed access, you can carry all the data you need on the road. Superb 4G/3G connection and easy-to-use routing functions make it a desirable router in pocket wherever you go.

Form Factor

  • Dimension: 73.6 x 53 x 24.3 (LxWxH, mm)
  • Weight (g): 90 g

Connectivity and Data Speed

  • B1 (2100 MHz), B2 (1900 MHz), B3 (1800 MHz), B4 (AWS), B5 (850 MHz), B7 (2600 MHz), Band 8 (900 MHz), B12 (700 MHz), B13 (700 MHz), Band 14 (700 MHz), B17 (700 MHz), B20 (EDD800 MHz), B25 (1900 MHz), Band 26 (800 MHz), B38 (2500 MHz), B40 (2300 MHz), Band 41 (2500 MHz) -- Multiple band sku combinations for world-wide usage and welcome for customized band sku discussion.
  • LTE Data Rate: DL up to 100 Mbps, UL up to 50 Mbps
  • LTE Bandwidth: up to 20 MHz
  • EVDO Rev.A: 800MHz/AWS/1900MHz fallback supported
  • UMTS/HSPA+: 850 MHz/900 MHz/AWS/1900 MHz/2100 MHz fallback supported
  • WLAN: 802.11b/g/n
  • WiFi On/Off Switch button: Embedded beneath battery

Device Interface

  • USB for PC connection: tethering connectivity to PC with driver supported
  • Power Switch: On/Off by sliding device top cover
  • Power Supply: Charging via Micro-USB, AC adaptor's input 100~240V, 50-60Hz, output 5V DC
  • Power State: Active/Sleep
  • Embedded Battery: 2,400 re-chargeable Li-Ion, D.C 3.7V
  • Wake-up/WPS Button: short press for device wake-up and long press (>5s) for WPS activation
  • Micro SD: support up to 32 Gbytes
  • Status Indication: LED signal
    1. Cellular strength
    2. Network status
    3. WiFi status
    4. Roaming status
    5. Battery power status/charging status
  • Operation Modes
    1. USB & WiFi simultaneous
    2. WiFi only
    3. USB only (WiFi can be turned off by hardware switch or by software switch)

Antenna

  • Cellular embedded main antenna
  • Cellular embedded diversity antenna
  • Cellular external main antenna port
  • Cellular external diversity antenna port
  • WiFi embedded antenna

Router Features

  • Security: Firewall, VPN (PPTP, IPSec, SSL) Passthrough, Port forwarding, Port filtering, DMZ, UPnP, ALGs
  • DHCP: DHCP Server/Client, DHCP Reservation
  • IPv4 and IPv6
  • WiFi: WPA2/WPA/AES/TKIP, WPA/WPA2 PSK mode, SSID Broadcast disable, WPS, Access Control (MAC filter), WLAN range control, Max number of user control, WiFi on/off SW button

SW Features

  • Over the air FW upgrade
  • Over the air Device Management
  • OS Support (for USB connection): Windows, MAC OS X, Linux Ubuntu
  • Connection Status: Network name, Signal strength, Roaming indication, Radio technology, Data volume and statistics, Number of user on WLAN
  • Connection Management: Auto Connection, Connection on demand, Auto APN matching with USIM, APN database update through browser-based GUI, APN profile
